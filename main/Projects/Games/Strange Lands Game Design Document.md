# Emergent Gameplay
## Systems
- Destruction
- Traps
	- Light and mirrors
- Movement
- Gravity
	- Modifiable and relative to some frame. Super Mario Galaxy map should be possible
- Explosions
- Weapons
- Scalable/Custom Characters
- Ragdolls and moveable ragdolls[^1]
- Procedural Animation[^2]
